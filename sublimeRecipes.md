## Column Editing

* Highlight Word

* Add Word:<br>

* Go to Previous Word:<br>

* Skip Current Word:<br>
ctrl+k+d

* Open Command Pallet [WIN]:<br>
ctrl+shift+p

* Open Help Search [MAC]:<br>
cmd?+shift+/

* Incremental Search
ctrl + i
search
